## 遇到的问题

项目部署时直接升级`node`版本，经常会遇到依赖库报错问题，比如`node-sass`、一些`npm Error`/`node Error`等

## 常用的处理方向

### 添加babel-runtime兼容语法

```shell
npm install babel-runtime
```

### 其他：npm-check检测依赖库版本

```shell
npm install -g npm-check
```

再在项目中使用以下命令检查更新
```shell
npm-check
```
