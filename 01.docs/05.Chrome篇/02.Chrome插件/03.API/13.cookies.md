# cookie
[官方文档](https://developer.chrome.com/docs/extensions/reference/cookies)

> 允许用户创建、获取、删除、查询cookie

```json
"permissions": [
  "cookies"
]
```

## 方法

### 增
```ts
chrome.cookies.set(
  details: object,
  callback?: function,
)
```

```js
chrome.cookies.set({
  url: 'https://www.baidu.com',
  name: 'name',
  value: 'value',
  domain: 'baidu.com',
  path: '/',
  secure: true,
  httpOnly: true,
  expirationDate: Date.now() + 1000 * 60 * 60 * 24 * 365 * 10,
}, (cookie) => {
  console.log('🍄  background: >>>>>>>>>>>>>>>>>> 创建cookie', Date.now(), cookie);
});
```

### 删
```ts
chrome.cookies.remove(
  details: object,
  callback?: function,
)
```

```js
chrome.cookies.remove({
  url: 'https://www.baidu.com',
  name: 'name',
}, (cookie) => {
  console.log('🍄  background: >>>>>>>>>>>>>>>>>> 删除cookie', Date.now(), cookie);
});
```

```ts
  details: object,
  callback?: function,

```
