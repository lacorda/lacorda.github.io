# first-child与first-of-type

[Demo](/site#/html-css/first-child)

## first-child

属满足两个条件:

1. 指定标签（没有指定则是全标签）
2. 其父元素的 ***第一个子元素是指定标签***（没有指定则是全标签）

## first-of-type

属满足两个条件:

1. 作用域内兄弟元素之间 ***同标签的第一个元素***
2. 必须是指定标签（没有指定则是全标签）
